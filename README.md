# SuperClaw Dashboard

The SuperClaw Dashboard is a web interface for monitoring and managing your OpenClaw AI companion.

## Installation

Install via the SuperClaw CLI:

```bash
npm install -g @skunkceo/superclaw-cli
superclaw install
```

## Updates

The CLI will notify you when updates are available. Run:

```bash
superclaw update
```
